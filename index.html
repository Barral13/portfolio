Principais Tecnologias

.NET

A plataforma .NET oferece um ambiente unificado para o desenvolvimento de diferentes tipos de aplicativos, como web, desktop, mobile e mais. Utilizando linguagens como C# é possível desenvolver com alta performance e segurança.

ASP.NET

ASP.NET é um framework robusto para desenvolvimento web que permite criar aplicações dinâmicas e seguras. Ele facilita a criação de APIs RESTful e integrações com diversas plataformas de forma eficiente.

Blazor

Blazor permite criar aplicações web interativas utilizando C# ao invés de JavaScript. Ele suporta renderização no servidor ou no cliente, trazendo flexibilidade e poder para desenvolvedores que preferem a linguagem .NET para o front-end.

Principais Projetos

Projeto de Loja Virtual

Barral Games é uma loja online fictícia desenvolvida para treinar habilidades em frontend usando HTML, CSS, JavaScript e Tailwind CSS. A loja exibe consoles modernos e retrô, como PlayStation 5 e Xbox Series X, em um layout responsivo e intuitivo. O design minimalista, feito com Tailwind, oferece uma navegação rápida e acessível, focando em aprendizado prático de conceitos de frontend.

Projeto de API para Blog

Esta API para blog é uma aplicação desenvolvida com .NET e ASP.NET, utilizando Entity Framework Core para gerenciar dados de forma eficiente. A segurança é implementada através da autenticação JWT, permitindo que usuários acessem funcionalidades de criação, leitura, atualização e exclusão de postagens de forma segura. Com uma arquitetura baseada em controllers, a API oferece endpoints claros e bem definidos, facilitando a integração com front-ends e outros serviços. O projeto suporta operações de gerenciamento de postagens e comentários, proporcionando uma experiência rica e interativa para desenvolvedores e usuários finais.
